# Справочник Радиолюбителя

## Описание

"Справочник Радиолюбителя" — это простое настольное приложение, созданное на Python с использованием библиотеки **Tkinter**. Оно предназначено для радиолюбителей и предоставляет быстрый доступ к информации о распространенных электронных компонентах, терминах и понятиях. Приложение позволяет искать определения, просматривать историю поисковых запросов, а также отображает полный список доступных терминов при наведении на специальную иконку.

## Основные возможности

*   **Поиск терминов:** Введите название компонента или термина в поле поиска, чтобы получить его описание. Поиск не чувствителен к регистру и поддерживает частичные совпадения.
*   **Отображение описаний:** Получайте четкие и лаконичные пояснения для каждого найденного термина.
*   **История поисков:** Приложение сохраняет последние введенные запросы, что позволяет быстро вернуться к ним.
*   **Интерактивный словарь:** Наведите курсор мыши на иконку 📖 (**справа от заголовка**), чтобы увидеть всплывающий список всех слов, доступных в базе данных.
*   **Удобный интерфейс:** Интуитивно понятный графический интерфейс делает работу со справочником простой и приятной.
*   **Кастомизация (частичная):** Возможность добавить файл `icon.ico` для иконки окна приложения.

## Как запустить приложение

1.  **Предварительные требования:**
    *   Установленный интерпретатор **Python 3.x** на вашем компьютере.
    *   Библиотека **Tkinter**. Обычно она входит в стандартную установку Python, но на некоторых дистрибутивах Linux может потребоваться установка (`sudo apt-get install python3-tk`).
2.  **Сохраните код:** Скопируйте весь код, предоставленный ранее, и сохраните его в файл с расширением `.py` (например, `radio_guide.py`).
3.  **Запустите из командной строки:**
    *   Откройте терминал (командную строку).
    *   Перейдите в директорию, где вы сохранили файл, используя команду `cd` (например, `cd Documents/PythonProjects`).
    *   Выполните команду:
        ```bash
        python radio_guide.py
        ```
4.  **Иконка окна (опционально):** Если вы хотите, чтобы иконка приложения в заголовке окна отображалась корректно, поместите файл с названием `icon.ico` в ту же папку, где находится скрипт Python. При отсутствии этого файла будет использована стандартная иконка Tkinter.

## Включенные данные

В приложении есть встроенная база данных, содержащая описания следующих терминов (список неполный):

*   Резистор (и его разновидности: постоянный, подстроечный, переменный)
*   Транзистор (биполярный)
*   Конденсатор
*   Диод
*   Индуктивность
*   Микросхема (ИС)
*   Операционный усилитель (ОУ)
*   Стабилизатор напряжения
*   Кварцевый резонатор
*   Плата (PCB)
*   Макетная плата (Breadboard)
*   ... и многие другие.

Данные жестко закодированы в классе `RadioHobbyistApp`.
